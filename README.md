# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. This project allows players to compete against the computer in the browser with a clean and responsive interface.

## Features
- Play Rock, Paper, or Scissors against the computer
- Instant result display after each round
- Score tracking for both player and computer
- Simple and beginner-friendly front-end design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder.
2. Launch the main HTML file in your browser.
3. Start playing the game.

## Project Purpose
This project was created as a fun front-end development exercise to practice HTML, CSS, and JavaScript interaction.
